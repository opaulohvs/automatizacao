# Comandos Completos para Commit e Push

## Comandos Sequenciais

```bash
git add .
git commit -m "Fix: Adicionar logs de debug e melhorar detecção de Chrome no Railway"
git push -u origin main
```

## Ou em uma linha (PowerShell)

```powershell
git add . ; git commit -m "Fix: Adicionar logs de debug e melhorar detecção de Chrome no Railway" ; git push -u origin main
```
